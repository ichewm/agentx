# Cursor Install

Install methods:

- `export-only`: write `.cursor/rules/<rule-id>.mdc` in an output folder.
- `path-copy`: copy `.cursor/rules/<rule-id>.mdc` into a target project.

Default destination:

```text
<target-project>/.cursor/rules/<rule-id>.mdc
```

The user must provide or confirm the target project path before installation.
