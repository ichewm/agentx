# Hermes Target Profile

Status: partially verified from Hermes Agent skill documentation.

Sources:

- `https://hermes-agent.nousresearch.com/docs/user-guide/skills/overview`
- `https://github.com/NousResearch/hermes-agent`

## Runtime Capabilities

- Native skills: supported.
- Rules: not generated by default.
- Prompts: supported as skill content.
- Commands: not generated by default.
- Tools: skills can define guidance and context; tool availability remains runtime-controlled.
- Connectors: not generated by default.
- Hooks: not generated by default.
- Subagents: not generated by default.

## Routing

- Runtime chooses capabilities: yes.
- Routing source: skill metadata and directory registration.

## Scopes

- Project: external skill directories may be configured by the runtime.
- User: `~/.hermes/skills/<skill-id>/`.
- Organization: not represented by this profile.
- Plugin: not represented by this profile.

## Refresh Requirements

Runtime benchmark must verify that Hermes discovers and routes to the skill.

## Unsupported Features

AgentX must not claim support for unverified Hermes extensions beyond skills.
