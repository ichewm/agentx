# OpenClaw Target Profile

Status: partially verified from OpenClaw documentation and repository.

Sources:

- `https://github.com/openclaw/openclaw`
- `https://docs.openclaw.dev/skills/overview`

## Runtime Capabilities

- Native skills: supported.
- Rules: not generated by default.
- Prompts: supported as skill instructions.
- Commands: not generated by default.
- Tools: skills can guide available OpenClaw behavior but do not install model tools.
- Connectors: not generated by default.
- Hooks: not generated by default.
- Subagents: not generated by default.

## Routing

- Runtime chooses capabilities: yes.
- Routing source: skill metadata and description.

## Scopes

- Project: supported through local workspace skill install.
- User: supported through global skill install.
- Organization: not represented by this profile.
- Plugin: not represented by this profile.

## Refresh Requirements

Runtime benchmark must verify installed skill discovery.

## Unsupported Features

AgentX must not claim OpenClaw support for unverified plugin, hook, or tool-extension mechanisms.
