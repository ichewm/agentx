# GitHub Copilot Review Checklist

- Verify `SKILL.md` metadata and description.
- Verify repository output uses `.github/skills/<skill-id>/`.
- Verify personal output uses `~/.copilot/skills/<skill-id>/`.
- Verify the artifact is scoped to Copilot coding agent unless another Copilot runtime is verified.
- Verify custom instructions are not mislabeled as skills.
