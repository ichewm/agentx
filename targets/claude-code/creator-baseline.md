# Claude Code Creator Baseline

Baseline: Claude Code skill package.

Source:

- `https://docs.anthropic.com/en/docs/claude-code/skills`

Minimum package:

```text
<skill-id>/
  SKILL.md
```

`SKILL.md` contains frontmatter metadata plus instructions. Bundled resources may be placed in sibling folders and loaded by the runtime when relevant.

AgentX must follow official Claude Code skill metadata, naming, and packaging guidance before claiming final Claude Code compatibility.
