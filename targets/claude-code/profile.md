# Claude Code Target Profile

Status: verified against Anthropic Claude Code skill documentation.

Sources:

- `https://docs.anthropic.com/en/docs/claude-code/skills`

## Runtime Capabilities

- Native skills: supported.
- Rules: not used as the primary target artifact in this profile.
- Prompts: supported as skill instructions and references.
- Commands: not generated by default.
- Tools: skills can instruct use of available Claude Code tools but do not install tools into the model.
- Connectors: not generated by default.
- Hooks: not generated by default.
- Subagents: not generated by default.

## Routing

- Runtime chooses capabilities: yes.
- Routing source: skill metadata and description.

## Scopes

- Project: `.claude/skills/<skill-id>/`.
- User: `~/.claude/skills/<skill-id>/`.
- Organization: not represented by this profile.
- Plugin: plugin distribution exists outside this profile and must follow official guidance before use.

## Refresh Requirements

Claude Code must discover the skill from its supported skill directory. Restart or refresh behavior must be verified during runtime benchmark.

## Unsupported Features

AgentX must not claim that a Claude Code skill grants tools, bypasses permissions, or changes model policy.
