# Claude Code Install

Install methods:

- `path-copy`: project install to `.claude/skills/<skill-id>/`.
- `path-copy`: user install to `~/.claude/skills/<skill-id>/`.
- `export-only`: leave generated files in AgentX output.

Default method: plan-only export. Ask the user to choose project or user scope before installation.

Installation requires explicit user confirmation and target-ready evidence.
