# Target Profiles

Each target runtime directory must contain:

- `profile.md`
- `creator-baseline.md`
- `package-layout.md`
- `install.md`
- `review-checklist.md`

Use `Unknown` for unverified facts in profile research. Unknown values may be used for internal drafts only. Final delivery must resolve, mark manual, mark unsupported, or explicitly scope out any Unknown that affects compatibility, routing, package layout, installation, safety, or benchmark validity.
