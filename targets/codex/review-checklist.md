# Codex Review Checklist

- Verify `SKILL.md` frontmatter has `name` and `description`.
- Verify description clearly scopes trigger conditions.
- Verify detailed content is progressively disclosed through references when needed.
- Verify no tool or permission is claimed unless the current Codex runtime exposes it.
- Verify generated files contain no unresolved `Unknown`, `TBD`, or `TODO`.
