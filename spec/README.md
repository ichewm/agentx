# AgentX Specification

This directory is the authoritative specification for AgentX.

AgentX is not a local registry control plane. It is an AI-native capability workbench: a set of meta-skills loaded into an agent runtime, plus a thin deterministic helper for filesystem delivery, backup, rollback, and verification.

English files in this directory are authoritative for AI agents and implementation work. Chinese files ending in `.zh.md` are human review translations only.

## Core Principle

The runtime chooses skills. AgentX helps the current AI create, translate, review, and package capabilities for other runtimes.
